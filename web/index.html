This template is a quick start for your electron projects. The main node script is in main.js. You can find the interface files inside the web directory. This directory includes directories for frameworks, images, resources, scripts and stylesheets. The main page is in the index.html file.

This template incorporates jquery, bootstrap, and popper.js.

To run the project, first install it:

$ npm install

Then run it:

$ npm start

Or also:

$ node .

To make the window border-less, set the frame property to false in main.js. You can later manipulate the window through your interface by adding some events and using the show, minimize, and restore functions (among others. See the Electron JS API docs).

To interact with the node interpreter from the interface, create a new event in your main.js file using the instance of ipcMain, then send an event from the interface using ipcRenderer. Example:

main.js 
ipcMain.on('buttonPress', (event,button) => {
  console.log("You pressed the button: " + button);
})
 On you interface's JS: 
$(".btn").click(function() {
  console.log("Value: " + $(this).val());
  ipcRenderer.send('buttonPress', $(this).val());
});

Once you are done with your electron program, on Windows, you can package it using something like electron-packager to package it into an executable and distribute it if you wish. You can create an installer with something like Inno Setup or NSIS.

This is just a template for a quick start, and has no license. Meaning that you can use it however you like.
